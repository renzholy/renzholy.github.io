Bitcoin & Blockchain

比特币与区块链

Bitcoin

比特币

两种含义：一种 blockchain ；一种 token。

Satoshi Nakamoto

中本聪

bitcoin 创造者对自己的称呼。

很多人自称是 bitcoin 创造者，但都拿不出证据。如果有个人能够使用中本聪在 genesis block 中的 address 1A1zP1eP5QGefi2DMPTfTL5SLmv7DivfNa 进行一次 transaction，并在此之前向大众公布该 transaction 的细节，那么就说明 TA 拥有这个 wallet 的 private key ，就能够证明 TA 是中本聪（或者是偷了中本聪的 wallet）。

White paper

白皮书

blockchain 的产品、技术文档

也有黄皮书、紫皮书等等，取决于封面用的什么颜色。

Blockchain

区块链

一种数据库，数据库中的数据对所有人可见，数据的量只增加不减少，历史数据无法被修改（除非遭受 51% attack）。

Block

区块

blockchain 存储数据的基本单位。

以 bitcoin 为例，平均每 10 分钟由某个 miner 算出一个 block，每个 block 中存储了：

  • 前一个 block 是哪个
  • 区块的高度/序号，在前一个 block 基础上 +1
  • 被哪个 miner 算出
  • 被算出来的时间点
  • 产出了多少新的币
  • 0 至若干个 transaction
  • gas
  • coinbase

等等信息，因为 block 的“前一个 Block 是哪个”信息，所以从最新的 block 可以追溯到第一个 block，连成一个链条，所以被称为 blockchain。第一个 block 没有前一个 block，被称为 genesis block。

Genesis block

创世区块

每个 blockchain 的第一个 block。

通常写在代码里或由环境变量指定。

Testnet

测试网络

测试环境，和主 blockchain 的 genesis block 不同，代码可能相同也可能不同。

Coinbase

比特币区块留言

miner 在 bitcoin block 里留的一段话，有长度限制。

著名交易所 Coinbase 的名字应该就是从这里来的（没有考证）。

Satoshi Nakamoto 在 genesis block 里留的话是：

EThe Times 03/Jan/2009 Chancellor on brink of second bailout for banks

在各种 explorer 中可以看到，比如 blockchair🔗。

bitcoin 里的称为 coinbase，其他 blockchain 不一定称为 coinbase。

Explorer

区块浏览器

用于浏览 blockchain 中的数据的 App，一般是网页。

基本上每个 blockchain 都有一个或多个官方或第三方提供的浏览器。

Gas

手续费

transaction 的发起者给 miner 的小费。

更多的小费可以加快 transaction ，但 miner 很闲的时候，可以少给或不给小费，仍然能够较快完成 transaction。

Fork

分叉

同时产生了两个或两个以上的 block ，它们指向了相同的前一个 block。

比如一个人生了两个孩子，怎么来继承遗产，有几种情况：

  • 区块冲突：其中一个夭折了，剩下一个继承所有遗产。
  • 硬分叉：两个孩子生死决斗，赢的继承所有遗产。
  • 软分叉：两个孩子比赛，谁厉害谁继承的多，输的也有份。也有可能势均力敌，几乎平分。

Transaction

交易/转账

使用 private key 进行签名，把 token 从自己的 address 转移到另一个 address 的过程。

Pending transaction

待确认交易

由于 transaction 发起者没给或少给 gas ，或者 miner 忙不过来导致的，还没有被 mining 的 transaction。

Token

代币

可流通、拆分的数字权益证明，去中心化共识的加密数字令牌。

每个 blockchain 有一个 native token 或称 coin，在其基础之上有多种 token。

可以把 coin 理解为货币，token 理解为 Q 币，NFT 理解为 QQ 秀皮肤。

并不是所有 token 都是去中心化的，一般 stablecoin 都是中心化的。

Wallet

钱包

进行 transaction 所必须的东西，分为 address 和 private key 两部分，这两部分是一一对应的。

Address

钱包地址/公钥

wallet 公开的部分，相当于银行卡卡号。

不同的是：所有人都可以看到你的（以及其他人的） address 里有多少 token，以及历史 transaction（一些拥有匿名交易功能的区块链除外）。

是随机生成的，不是某个中央机构发放的，并且是匿名的。

如果一个 address 没有任何 transaction 或 token，那就没有人能知道它的存在。

Private key

私钥

wallet 私密的部分，相当于银行卡密码。

如果忘记了，永远找不回来。

Mnemonic Phrase

助记词

使用 12 个单词来描述 private key，更方便记忆。和 private key 有同等作用，是一一对应的关系。

Hardware Wallet

硬件钱包

类似于网银盾。

Layer2

二层网络/闪电网络

附属于某个或多个 blockchain 的 blockchain。

如果把 bitcoin 当成央行，layer2 就是地方分行。

bitcoin 平均 10 分钟才产生一个 block，太慢了，所以在 layer2 上完成频繁的交易，再在 bitcoin 上进行汇总信息。

Mining

挖矿

miner 把 transaction 打包放进 block 中，并完成 POW 的过程。

有些 blockchain 不是 POW，而是采用其他证明方式。

Hash rate

算力

进行 POW 时的计算能力。

POW

Proof of work，工作量证明

一种 miner 证明自己挖出了某个 block 的方式。

让计算机算难题，哪台计算机最先算对就有了工作量证明，算错的或者晚了一步的就白算。

bitcoin 的计算难度越来越大，参与的人越来越多，最开始用 CPU 算，后来用 GPU 算，再后来用专用的硬件来算，这里统称为计算机。

不是所有 blockchain 都是使用工作量进行证明。

POW 费电。

Miner

矿工

计算难题的计算机，也可以指用计算机算难题的人。

Mining pool

矿池

一个计算机太慢，多个计算机组成超算就是矿池，吃大锅饭。

矿池一般是中心化运营，运营者会从中抽成，再按 hash rate 的比例分成。

Mining revenue

挖矿收益

由两部分组成：block 中新产生的 bitcoin 和 transaction 里的 gas

51% attack

51% 算力攻击

用任何手段掌握半数以上的 miner 的 hash rate，就可以篡改 bitcoin 的历史 block。

超过 50% 即可，无需到 51%。

不是每个 blockchain 都会遭到 51% 攻击。
